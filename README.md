# Streamlit CRUD App with MySQL

This is a Streamlit-based web application for managing data through CRUD operations (Create, Read, Update, Delete) in a MySQL database. It provides a user-friendly interface for efficient data manipulation directly from your browser.

## Features

- **Create**: Add new records.
- **Read**: View and search through existing data.
- **Update**: Modify existing records.
- **Delete**: Remove unwanted records.

## Tech Stack

- **Streamlit** for the web interface.
- **MySQL** for data storage.

## Getting Started

To get this application up and running on your local machine, follow these steps:

1. **Clone this repository**

    ```
    git clone [Copy Repository URL]
    ```

2. **Ensure you have Python and MySQL installed**
3. **Install dependencies**

    Navigate to the project directory and run:

    ```
    pip install -r requirements.txt
    ```

4. **Add database**
    1. Go to MySQL and create database. Name: "crud_new1"
    2. Import the ```uses.sql``` file to the database
    
5. **Start the app**

    ```
    streamlit run crud.py
    ```

6. **Access the web interface**

    The application will be accessible via the URL provided by Streamlit.
